Project Name : Recipe Code Poe Part1

This is a command line application that allows the user to enter and store 
the ingredients and steps for one recipe.

To run the program : 
Step 1 run the C# code on Visual Studio
Step 2 Run the main program  ExamplePoe.sln
Step 3 Start by creating the Recipe
Step 4 Choose a scale you would want to use
Step 5 Display the Recipe you created
Step 6 IF not pleased reset the Scale and reenter a new one
Step 7 Clear the recipe for a new one
Step 8 Exit the program.
